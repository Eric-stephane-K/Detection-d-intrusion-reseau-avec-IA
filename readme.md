# Système Intelligent de Détection d’Intrusions Réseau

## Présentation
Ce projet porte sur la conception et le développement d’un système intelligent de détection d’intrusions réseau (NIDS) basé sur le Machine Learning. L’objectif principal était d’analyser le trafic réseau afin d’identifier et classifier les activités malveillantes en temps réel.

Le système exploite plusieurs algorithmes d’apprentissage supervisé afin d’améliorer la précision de détection et d’évaluer les performances des modèles grâce à différentes métriques d’analyse.

---

## Objectifs
- Détecter les activités malveillantes sur un réseau
- Classifier les différents types d’intrusions
- Comparer les performances de plusieurs algorithmes de Machine Learning
- Visualiser les résultats de détection à travers une interface web

---

## Technologies utilisées
- **Python**
- **Scikit-learn**
- **Pandas / NumPy**
- **Matplotlib / Seaborn**
- **Flask**

### Algorithmes de Machine Learning
- Régression Logistique
- K-Nearest Neighbors (KNN)
- Arbre de Décision
- Forêt Aléatoire (Random Forest)

---

## Fonctionnalités
- Prétraitement et nettoyage des données réseau
- Sélection et normalisation des caractéristiques
- Entraînement et évaluation des modèles
- Système de prédiction des intrusions
- Visualisation des résultats en temps réel
- Intégration d’une interface web avec Flask

---

## Évaluation des modèles
Les performances des modèles ont été évaluées à l’aide de :
- Matrices de confusion
- Accuracy Score
- Précision / Recall
- Analyse des courbes ROC

---

## Architecture du projet
```text
Dataset → Prétraitement des données → Entraînement des modèles → Évaluation → Prédiction → Interface Web Flask
```

---

## Résultats
Le projet a démontré l’efficacité des algorithmes de Machine Learning dans la détection et la classification des intrusions réseau. Les modèles Random Forest et Arbre de Décision ont obtenu de très bonnes performances avec une précision élevée et une classification fiable des connexions malveillantes.
